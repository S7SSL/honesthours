## Quick Install (30 seconds)

### Option 1: Download & Drag-Drop (Easiest)

1. **Download** the extension file:
   - [honesthours-extension-v1.2.zip](honesthours-extension-v1.2.zip)

2. **Unzip the file**
   - Right-click → "Extract All" (Windows)
   - Double-click → "Extract" (Mac)
   - Creates folder: `extension/`

3. **Open Chrome Extensions Page**
   - Type in address bar: `chrome://extensions/`
   - Press Enter

4. **Enable Developer Mode**
   - **Top right** of page
   - Toggle **"Developer mode"** ON (blue)

5. **Load the Extension**
   - Click blue **"Load unpacked"** button
   - Select the **`extension/`** folder
   - Click **"Select Folder"**

6. **Done!**
   - Extension icon appears in toolbar
   - Click it to start tracking

---

### Option 2: Manual Install from Folder (Alternative)

If you prefer not to use ZIP:

1. Clone the repository or download the source
2. Navigate to: `honesthours/extension/`
3. Follow steps 3-6 above

## Verify Installation

### Check Extension Loaded
- **Top right corner** of Chrome
- You should see "Honest Hours" icon (clock + checkmark)
- Click it → popup appears with "Start" button

### Check Permissions
- On `chrome://extensions/` page
- Find "Honest Hours"
- It should show:
  - ✅ storage
  - ✅ alarms
  - ✅ idle

### Test It Works
1. Click extension icon
2. Click **"Start"** button
3. Wait 5 seconds
4. Timer should show: `0h 00m 05s`
5. Click **"Stop"**
6. Time persists

## Troubleshooting

### "Load unpacked" button is greyed out
- Make sure **Developer mode** is ON (toggle blue in top right)

### Popup doesn't appear
- **Reload the page** (F5)
- **Restart Chrome** completely
- **Re-add the extension** via "Load unpacked"

### Timer doesn't count
- Open DevTools (F12)
- Check **Console** tab for errors
- Take screenshot of error and contact support

### Data not saving
- Check if **storage permission** is granted
- Go to `chrome://extensions/` → "Honest Hours" → check permissions
